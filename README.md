## 🚀 How to use this Homebrew Tap?

This repository serves as my personal [Homebrew Tap](https://docs.brew.sh/Taps) for distributing custom macOS applications that I have developed.

To install any of the applications listed below, you first need to add this repository to your Homebrew taps:

```bash
brew tap muhammadsaif7717/apps
```

*(Note: Depending on your macOS and Homebrew configuration, you might also need to run `brew trust muhammadsaif7717/apps`)*

## 📦 Available Applications (Casks)

### ☁️ [G Drive Connector](https://github.com/muhammadsaif7717/GDriveConnector)
A modern, seamless, and multi-account Google Drive desktop client featuring dynamic storage monitoring, unified trash view, and native OS-like file management. Built with Node.js and Electron.

**Installation:**
```bash
brew install --cask gdrive-connector
```

*Note: If the app fails to open after installation due to macOS Gatekeeper restrictions, run the following command to remove the quarantine attribute:*
```bash
sudo xattr -cr "/Applications/G Drive Connector.app"
```
